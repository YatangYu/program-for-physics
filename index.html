Program for Physics

  本網頁是配合中興物理系普通物理課程所設計，希望培養同學們善用程式解決物理問題的能力。

學物理為什麼要寫程式?

  物理的領域，涵蓋的範圍相當廣，因此能研究的問題也相當廣泛。
  在基礎的課程中，我們學了很多理論，也以這些理論為基礎，練習過很多計算問題。
  然而，我們能用手算解決的問題只是少數。
  大多數的問題，都不是能簡單算出來的，即使你有再簡單的模型假設，再厲害的數學技巧，
有些問題解不出來就是解不出來。

  因此，我們需要藉由電腦程式來幫我們解決這些問題。
  以電腦來解決問題有以下好處:
  1. 可執行龐大計算
  2. 即使模型複雜，也可以計算，可以更完整考慮模型
  3. 相較於統計軟體有更高的彈性

如何開始

  我們選擇python作為作業的程式語言，原因有二，第一個是好入門；
再來，因為全世界非常多人使用python，致使在python裡有相當多的工具是先人已經幫我們寫好的，我們可以直接套用。
  如果你還沒安裝python請參考下面連結跟著下載、安裝以及熟悉我們的編程的介面，接著就可以開始課程囉!
  • Python 下載、安裝 (Anaconda)
  • 應用程式簡介 (Spyder) 建議搭配 程式基本功(一) 一起看

作業分派

  • 作業分為基本程式能力練習以及物理應用兩部分
  • 程式能力練習作業有三份，為個人作業
  • 物理應用作業共有六份，每份含有進階問題與程式專題兩部分
  • 進階問題作業，為個人作業
  • 程式專題，為分組作業，至多三人一組
